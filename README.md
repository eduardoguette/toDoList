### Instrucciones de intalación y configuración

Clona este repositorio. Necesitaras `node` y `npm` instalados en tu ordenador.

Instalación:

`npm install`

Ejecutar un test:

`npm test run`

Inicia el servidor:

`npm start`

Visita la app:

`localhost:3000/`
